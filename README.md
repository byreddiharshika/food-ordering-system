#Food Ordering System

#Description
The Food Ordering System is a full-stack web application that allows users to browse food items, add them to a cart, and place orders online. It provides a simple and user-friendly interface for seamless ordering.

#Features
- User registration and login authentication
- Dynamic food menu display
- Add to cart and order placement
- Database integration for storing user and order details

#Technologies Used
- Python
- Django
- HTML, CSS
- MySQL
